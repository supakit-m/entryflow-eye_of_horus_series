# Demo Video

Place a demo video file named `demo.mp4` in this directory (`public/demo/demo.mp4`).

## Requirements
- Format: MP4 (H.264 recommended for broad browser support)
- Content: Footage of people walking through an entrance or corridor
- Recommended resolution: 640×480 or 1280×720
- Recommended duration: 30–120 seconds (will loop automatically)

## Suggested Sources
- Record your own walkthrough footage
- Use any royalty-free crowd/entrance footage
- Pixabay, Pexels, or Coverr for free stock video

## Notes
- The demo mode uses the exact same detection pipeline as live camera mode
- No camera permission is required in demo mode
- Video is processed entirely client-side; nothing is uploaded
